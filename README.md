# 📚 Revistas de Biblioteconomia

Portal acadêmico dinâmico projetado para monitorar e consolidar as principais revistas científicas brasileiras e internacionais de **Biblioteconomia e Ciência da Informação**.

## 🚀 Funcionalidades
- **Radar de Pesquisa 7 Dias:** Monitoramento automatizado de novas publicações via feeds RSS/OJS.
- **Catálogo Geral:** Lista exaustiva de periódicos com link direto, ISSN, Instituição e Qualis.
- **Interface Premium:** Design institucional sóbrio e otimizado para alta densidade de informação.
- **Single-File:** Todo o sistema (HTML, CSS e JS) está contido em um único arquivo para fácil portabilidade.

## 🛠️ Tecnologias Utilizadas
- HTML5 Semântico
- CSS3 (Flexbox/Grid/Variables)
- Vanilla JavaScript
- Font Awesome Icons
- Google Favicons API (Integração de Logos)
- RSS2JSON API (Proxy de Raspagem)
