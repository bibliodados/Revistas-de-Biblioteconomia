# Revistas de Biblioteconomia
Portal unificado para acesso rapido a periodicos cientificos da area de Ciencia da Informacao e Biblioteconomia.

## Funcionalidades
Monitoramento 24h: Radar que identifica publicacoes dos ultimos 7 dias.
Catalogo Unificado: Acesso rapido as principais revistas da area (PCI, IBICT, Transinformacao, etc.).
Design de Alta Densidade: Visualizacao otimizada para consulta rapida de dados.

## Tecnologias
HTML5 / CSS3 (Design Responsivo Premium)
JavaScript Vanilla
Integracao via Feeds RSS e OJS Gateway
