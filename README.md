# Radar de Biblioteconomia

Portal academico dinamico projetado para monitorar e consolidar as principais revistas cientificas brasileiras e internacionais de Biblioteconomia e Ciencia da Informacao.

## Funcionalidades
- Monitoramento 24h: Radar que identifica publicacoes dos ultimos 7 dias.
- - Catalogo Unificado: Acesso rapido as principais revistas da area (PCI, IBICT, Transinformacao, etc.).
  - - Design de Alta Densidade: Visualizacao otimizada para consulta rapida de dados.
